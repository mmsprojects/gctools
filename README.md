## Sobre o Projeto

O **GC Tools** é um plugin para o QGIS que foi projetado para auxiliar em tarefas de IA (Inteligência Artificial) que envolvem dados geoespaciais. Este plugin fornece funcionalidades avançadas para amostragens com data augmentation, predições e validações, tornando-o uma ferramenta essencial para análise geoespacial avançada.
